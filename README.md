## ðŸ“– Introduction

**Daily Momentum** is a gamified task management application designed to visualize your daily achievements and motivate personal growth.

Inspired by the progression systems found in RPG games (like *Marvel Future Fight*), this app embodies the philosophy that consistent completion of daily "missions" leads to a powerful future. Just as game characters grow by clearing daily quests, users build their own momentum by completing daily tasks.

## âœ¨ Key Features

* **Daily Missions:** Manage your daily tasks as "missions" to clear, gamifying the to-do list experience.
* **Victory Celebration:** When all daily missions are completed, the app rewards the user with a celebratory audio-visual effect (Confetti & Boom sound) to reinforce positive habits.
* **Weekly Growth Log:** Visualizes consistency with a colorful weekly bar chart. Watch your past efforts pile up to create a "streak" of success.
* **Category Colors:** Customize tasks with distinct colors for Development, Study, Health, and Life to create a vibrant activity log.
* **Dark Mode UI:** A sleek, immersive dark mode interface designed to enhance focus and reduce eye strain.
* **Data Privacy:** All data is persisted locally using the browser's `localStorage`, ensuring user privacy without the need for a backend server.
